# 01 · 测试用例

> 范围:PaddleCppAPITest 仓库下 `test/` 目录的 GoogleTest 测试用例贡献。22 个 PR 中**约 16 个**是测试用例 PR(或测试 + 文档混合)。

## 1. 主题概述

测试仓的工作流是:

1. 编写 **一份** GoogleTest 代码,只使用 `at::` / `c10::` / `torch::` 命名空间下的接口。
2. 通过 CMake 选项 `TORCH_DIR=...` 切换 link 目标:
   - `TORCH_DIR=<paddle build>` → 链接 Paddle compat 层
   - `TORCH_DIR=<libtorch install>` → 链接真正的 libtorch
3. 测试运行后输出 `paddle_<TestName>.log` 和 `torch_<TestName>.log`,用 `result_cmp.sh` 做 `diff`。
4. `diff` 为空 = MATCH;非空 = mismatch,需要 fix compat 层或在 mismatch 表中记录。

测试代码必须**严格只用对外的 libtorch API**,不能依赖 Paddle 内部类型。
